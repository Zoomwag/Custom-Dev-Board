# RP2040 Development Board



---

## Overview

This dev board is based on the Raspberry Pi RP2040 microcontroller that i deigned to learn how MCUS work
---

##  Features

- RP2040 dual-core ARM Cortex-M0+ microcontroller
- USB C interface for power and programming (much better than the picos micro usb)
- GPIO breakout for easy access
- On-board voltage regulation
- Compact and clean PCB layout
